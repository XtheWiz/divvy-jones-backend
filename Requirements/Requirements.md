# Requirements

1.  Sign Up Method
    1.  Email / Password with Verify Email
    2.  Google
    3.  Apple ID
2.  ผู้ใช้สามารถสร้าง Trip Group ได้
3.  สร้าง / เข้าร่วมกลุ่ม
    - **หน้าสร้างกลุ่ม**
        - หัวเรื่อง: "สร้างกลุ่มแบ่งบิล"
        - ช่องกรอก: _ชื่อกลุ่ม_, _ป้ายโต๊ะ/รอบ (ไม่บังคับ)_
        - ปุ่ม: **สร้างกลุ่ม**
        - หลังสร้าง: แสดงดังนี้
            - **QR Code กลุ่ม** (ขนาดใหญ่)
            - รหัสเข้าร่วม 6 ตัวอักษร
            - ปุ่มแชร์
        - ข้อความแจ้ง: "คนอื่นสแกนหรือใส่รหัสเพื่อเข้าร่วม"
    - **หน้าเข้าร่วมกลุ่ม**
        - แท็บ: **สแกน QR** (มุมมองกล้องพร้อมกรอบสี่เหลี่ยม) / **ใส่รหัส** (ช่องกรอกข้อความ + ปุ่ม **เข้าร่วม**)
        - เมื่อสำเร็จ: แสดงห้องรอกลุ่ม
    - **ห้องรอกลุ่ม**
        - หัวเรื่อง: "{ชื่อกลุ่ม} – รอบที่ {X}"
        - ป้ายผู้สร้างบนผู้ใช้ A
        - Chip ผู้เข้าร่วม: อวาตาร์ + ชื่อย่อ
        - แบนเนอร์ใต้ Chip: "คนที่เข้าร่วมผ่าน QR/รหัสจะปรากฏที่นี่"
        - แถวปุ่ม: **เพิ่มรายการ**, **เพิ่มการปรับ %**, **ค่าบริการ**, **VAT**
        - รอง: **ตั้งค่าผู้จ่าย**, **ดูสรุป**
